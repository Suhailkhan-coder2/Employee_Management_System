# Employee Management System

**Employee Management System** is a **Django-based web application** that allows organizations to manage employees efficiently. 

---

## Features
- Add / View / Edit / Delete Employees (Admin only for Delete)
- User Authentication: Login & Registration
- Role-Based Access Control: Admin vs Staff
- Search Employees by Name
- Dashboard: Total employees and salary statistics
- Responsive UI using Bootstrap 5
- Secure Logout

---

## Technologies Used
- Backend: Python 3, Django 6
- Frontend: HTML, CSS, Bootstrap 5
- Database: SQLite (default)
- Deployment: Render / Any cloud platform
